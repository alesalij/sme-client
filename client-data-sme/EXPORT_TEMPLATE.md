# Шаблон файла для массовой выгрузки

## Формат файла

Поддерживаемые форматы:
- **.xlsx** - Microsoft Excel 2007+
- **.xls** - Microsoft Excel 97-2003
- **.csv** - Comma Separated Values (с разделителем ";")

## Структура файла

Файл должен содержать заголовки в первой строке:

| Заголовок | Обязательный | Описание | Пример |
|-----------|--------------|----------|--------|
| ИНН | Нет | ИНН клиента (10 цифр для ЮЛ, 12 для ИП) | 7713123456 |
| ОГРН | Нет | ОГРН клиента (13 цифр для ЮЛ, 15 для ИП) | 1027700132195 |
| Счет | Нет | Номер счета клиента | 40702810123456789012 |
| Краткое наименование | Нет | Название компании (для ЮЛ) | ООО "РОМАШКА" |
| ФИО | Нет | ФИО предпринимателя (для ИП) | Иванов Иван Иванович |

## CSV формат

При использовании CSV формата используйте разделитель ";" (точка с запятой):

```csv
ИНН;ОГРН;Счет;Краткое наименование;ФИО
7713123456;;40702810123456789012;;
123456789012;;;Иванов Иван Иванович
;1027700132195;;ООО "РОМАШКА";
```

## Правила заполнения

1. **Обязательные поля**: Хотя бы одно поле должно быть заполнено для каждой строки
2. **Формат ИНН**:
   - 10 цифр для юридических лиц
   - 12 цифр для индивидуальных предпринимателей
3. **Формат ОГРН**:
   - 13 цифр для юридических лиц
   - 15 цифр для индивидуальных предпринимателей
4. **Формат счета**: 20-25 цифр
5. **Кодировка**: UTF-8
6. **Разделитель**: Для CSV используйте ";"

## Ручной ввод данных

При ручном вводе данных в интерфейсе используйте следующий формат:

```
ИНН: 7713123456
ИНН: 123456789012, Счет: 40817810099910004312
ОГРН: 1027700132195, Краткое наименование: РОМАШКА
ФИО: Иванов Иван Иванович
```

## Валидация данных

Система автоматически проверяет:
- Корректность ИНН (длина и контрольная сумма)
- Корректность ОГРН (длина и контрольная сумма)
- Корректность номера счета
- Соответствие типа клиента (ЮЛ/ИП) указанным данным

## Ограничения

- **Максимальное количество записей**: 10 000 клиентов
- **Максимальный размер файла**: 10 MB
- **Таймаут обработки**: 5 минут

## Советы

1. **Используйте уникальные идентификаторы**: ИНН или ОГРН дают наиболее точные результаты
2. **Избегайте дубликатов**: Повторяющиеся записи будут обработаны только один раз
3. **Проверьте данные перед загрузкой**: Убедитесь в корректности форматов
4. **Используйте примеры**: Приложение содержит примеры для тестирования
5. **Мониторинг прогресса**: Следите за прогрессом выгрузки в реальном времени

## Результат выгрузки

Результат будет содержать:
- Входные данные из файла
- Дополнительные поля согласно выбранным настройкам
- Статус обработки (успех/ошибка)
- Информацию о найденных клиентах

Файл результата скачивается в формате Excel (.xlsx).
